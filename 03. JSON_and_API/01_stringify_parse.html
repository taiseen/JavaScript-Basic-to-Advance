JSON

JavaScript Object Notation

Server থেকে ৯৯% Data আসবে JSON Format এ (মানে string Formate এ)।

So Server থেকে Send করা Object এর সকল key:value গুলা String Format এ থাকার কারনে ওই Object এর key:value কে আলাদা করা সম্ভভ হয়ে উঠে না বা অনেক কষ্ট করতে হয়।

So কষ্ট থেকে রেহাই পেতে হলে use করতে হয় .parse()

Server থেকে পাঠানো Object এর পরে .parse() use করলে ওই string data টা কে JSON Object এ convert করে দিবে।
